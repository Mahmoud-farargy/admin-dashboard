## Getting Started

1. Clone project
    ```
    git clone https://github.com/Mahmoud-farargy/admin-dashboard.git
    ```
    
2. Install dependencies

   ```npm
   install
   ```

3. Fire up the server and watch files

   ```bash
   yarn dev
   ```

## Deployment

1. Start app
   ```
   npm run start
   ```
2. Test

   ```
   npm test
   ```

3. Build
   ```
   npm run build
   ```
4. Deploy!

   ```
   firebase deploy
   ```
